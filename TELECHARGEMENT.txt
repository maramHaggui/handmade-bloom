🚀 UTILISATION RAPIDE

ÉTAPE 1 : Décompressez
─────────────────────
Extrayez le fichier handmade-bloom-final.tar.gz

ÉTAPE 2 : Créez le dossier images
─────────────────────────────────
Créez un dossier nommé "image" dans le même répertoire

ÉTAPE 3 : Ajoutez vos images
────────────────────────────
Placez vos images de produits dans le dossier "image"
Les images doivent être nommées :
  - b1.jpg à b10.jpg    (bijoux)
  - c1.jpg à c13.jpg    (bougies)
  - a1.jpg à a16.jpg    (accessoires)
  - logo.png, owners.png (assets)

ÉTAPE 4 : Ouvrez index.html
───────────────────────────
Ouvrez simplement le fichier index.html dans votre navigateur

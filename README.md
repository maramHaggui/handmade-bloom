# Handmade Bloom - E-commerce Boutique

Une boutique en ligne élégante et moderne pour la vente de produits handmade (bijoux, bougies, accessoires).

## 📋 Description

Handmade Bloom est un site e-commerce complet avec :
- Page d'accueil avec présentation des produits
- Système de panier
- Authentification utilisateur
- Tableau de bord administrateur
- Gestion des produits et commandes
- Système d'avis clients

## 🚀 Installation

1. Téléchargez et extrayez le dossier ZIP
2. Créez un dossier `image` dans la racine du projet
3. Placez vos images de produits dans le dossier `image/`
4. Ouvrez `index.html` dans votre navigateur
